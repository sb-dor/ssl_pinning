Как взять SSL-сертификат backend-а

Для backend:

https://example.web.com

1. Скачать сертификат в папку Downloads

macOS / Linux

openssl s_client -showcerts \
-servername example.web.com \
-connect example.web.com:443 </dev/null 2>/dev/null \
| openssl x509 -outform PEM > ~/Downloads/api_cert.pem

После выполнения файл появится тут:

~/Downloads/api_cert.pem

Windows PowerShell

openssl s_client -showcerts `
-servername example.web.com `
-connect example.web.com:443 < $null 2>$null `
| openssl x509 -outform PEM > "$HOME\Downloads\api_cert.pem"

Файл появится тут:

C:\Users\YOUR_USER\Downloads\api_cert.pem

2. Добавить сертификат в Flutter project

Перемести файл сюда:

project/
 └── assets/
      └── certs/
           └── api_cert.pem

Добавь в pubspec.yaml:

flutter:
  assets:
    - assets/certs/api_cert.pem

3. Проверить время жизни сертификата

Если сертификат уже скачан:

openssl x509 -in ~/Downloads/api_cert.pem -noout -dates

Пример результата:

notBefore=May 10 00:00:00 2026 GMT
notAfter=Aug 8 23:59:59 2026 GMT

Главное поле:

notAfter

Это дата окончания сертификата.

4. Проверить срок сертификата сразу с сервера

openssl s_client -connect example.web.com:443 -servername example.web.com </dev/null 2>/dev/null \
| openssl x509 -noout -dates

5. Посмотреть всю информацию о сертификате

openssl x509 -in ~/Downloads/api_cert.pem -text -noout

Там будет:

  • issuer
  • domains
  • public key
  • fingerprints
  • срок жизни
  • SAN
  • algorithms

6. Использование в Flutter

import 'dart:io';

import 'package:flutter/services.dart';
import 'package:http/http.dart' as http;
import 'package:http/io_client.dart';

Future<http.Client> createPinnedHttpClient() async {
  final sslCert = await rootBundle.load('assets/certs/api_cert.pem');

  final securityContext = SecurityContext(withTrustedRoots: false);

  securityContext.setTrustedCertificatesBytes(
    sslCert.buffer.asUint8List(),
  );

  final httpClient = HttpClient(
    context: securityContext,
  );

  return IOClient(httpClient);
}

7. Запрос через pinned client

final client = await createPinnedHttpClient();

final response = await client.get(
  Uri.parse('https://example.web.com'),
);

print(response.body);

Важно

Если сертификат backend-а истечет или будет заменен, приложение с pinning на конкретный .pem может перестать делать запросы.

Для production лучше использовать pinning по public key / SPKI hash и заранее добавлять backup pin.
